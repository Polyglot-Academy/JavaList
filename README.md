# Projeto Java List

### 📌 Sobre o Projeto

O Java List é uma aplicação web desenvolvida em Java e JSP para auxiliar usuários na organização de tarefas do dia a dia. 

</br>

### 🎯 Objetivo

Criar um gerenciador de tarefas que facilite a organização pessoal e profissional, aumentando a produtividade dos usuários. 

</br>

### 🏗️ Escopo

O projeto inclui as seguintes funcionalidades:

- Cadastro de tarefas (título, descrição e categoria);
- Edição e exclusão de tarefas existentes;
- Classificação por categorias (trabalho, estudo, lazer, etc.);
- Persistência de dados em um banco de dados relacional.
- Recursos fora do escopo: Integração com outros sistemas, notificações automáticas e colaboração entre múltiplos usuários.

</br>

### 🔨 Tecnologias Utilizadas

- Back-end: Java, JSP, Servlets
- Banco de Dados: MySQL
- Front-end: HTML, CSS
- Controle de Versão: Git & GitHub

</br>

### 📅 Metodologia e Cronograma
A equipe utilizará a metodologia ágil, com entregas incrementais e testes contínuos.

- Mês 1: Definição dos requisitos e planejamento do desenvolvimento.
- Mês 2 a 4: Implementação das funcionalidades principais.
- Mês 5: Testes e correção de bugs.
- Mês 6: Implantação e entrega final.
